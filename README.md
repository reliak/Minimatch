#Minimatch
A minimal matching utility.

This is a port of SLaks' [Minimatch](https://github.com/SLaks/Minimatch) to
support DNX.

You can find the NuGet package (including installation notes) [here](https://www.nuget.org/packages/Minimatch.DNX/).

For usage instructions, please visit [Minimatch](https://github.com/SLaks/Minimatch).
